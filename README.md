💡 Proposta de valor

O aplicativo ReCiclo tem como principal benefício facilitar a conexão entre pessoas que desejam doar objetos e aquelas que precisam deles. A plataforma promove a reutilização de itens, reduz o desperdício e incentiva práticas sustentáveis, tornando o processo de doação simples, rápido e acessível.

📱 Descrição das telas
🔐 Tela de Login/Cadastro

Permite que o usuário acesse o aplicativo inserindo seus dados ou criando uma nova conta.

🏠 Tela de Feed

Exibe uma lista de itens disponíveis para doação próximos à localização do usuário. Possui barra de busca e filtros por categoria e distância.

📦 Tela de Detalhe do Item

Mostra informações completas do item selecionado, como foto, descrição, categoria e localização. Possui botão para demonstrar interesse.

💬 Tela de Chat

Permite a comunicação entre doador e interessado para combinar detalhes da retirada do item.

➕ Tela de Cadastro de Item

Permite ao usuário cadastrar um novo item para doação, adicionando foto, descrição e categoria.

🏆 Tela de Gamificação (Selo Eco)

Exibe uma recompensa simbólica após a conclusão de uma doação, incentivando o uso contínuo do aplicativo.

🔄 Jornada Principal do Usuário

Ao abrir o aplicativo, o usuário realiza o login ou cadastro. Em seguida, ele acessa a tela principal (feed), onde visualiza os itens disponíveis para doação próximos à sua localização.

O usuário pode utilizar a busca ou filtros para encontrar um item específico. Ao selecionar um item, ele é direcionado para a tela de detalhes, onde visualiza mais informações.

Caso tenha interesse, o usuário inicia uma conversa com o doador por meio do chat para combinar a retirada. Após a conclusão da doação, o usuário recebe um “Selo Eco” como forma de reconhecimento.
